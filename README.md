# AI Resume Builder

An AI-powered Resume Builder designed to help users create professional, ATS-friendly resumes effortlessly.

## Features
- AI Resume Summary Generation
- ATS Score Analysis
- Job Description Matching
- Multiple Resume Templates
- Live Resume Preview
- PDF Export
- Dark Mode
- Responsive Design

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- AI API
